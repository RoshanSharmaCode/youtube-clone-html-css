# 🎬 YouTube Clone (HTML & CSS)

A simple YouTube homepage clone built using HTML and CSS.  
This project focuses on layout design, UI structure, and responsiveness.

## 🚀 Features
- YouTube-like homepage layout
- Video thumbnails with titles & info
- Sidebar navigation
- Header with search bar
- Clean and responsive design

## 🛠️ Tech Stack
- HTML5
- CSS3

## 📂 Project Structure
- index.html
- style.css
- assets/

## ▶️ How to Run
1. Download or clone the repo
2. Open `index.html` in your browser

## 💡 Learning
- Flexbox & layout design
- Positioning elements
- Building real-world UI
